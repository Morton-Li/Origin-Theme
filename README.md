# Origin

Origin 是一个面向写作的 WordPress 主题，追求简约、优雅、克制的视觉体验。以清晰的文章列表、舒展的正文排版和轻量的交互为核心，让读者把注意力留给内容本身。

## 特性

- 简洁的首页文章流与精选文章展示。
- 舒适的文章页排版，适合长文阅读。
- 响应式导航、搜索、登录与注册体验。
- 支持深色模式与本地字体资源。
- 可配置 Cloudflare Turnstile 安全验证。

## 技术基线

- WordPress：7.0
- PHP：8.5 或更高版本

## 使用

下载发布包后，在 WordPress 后台上传并启用主题即可。主题启用后，可在外观自定义中调整导航布局与安全验证配置。
